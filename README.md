## 💥 WHAT IS THIS?

SINGULARITY DELTA IS A NEXT-GENERATION SYSTEM VERIFICATION ENGINE THAT ANALYZES DECISION-BASED SYSTEMS WITH MILITARY-GRADE PRECISION. IT VALIDATES YOUR ARCHITECTURAL DECISIONS, SYSTEM CONSTRAINTS, AND STRUCTURAL INTEGRITY LIKE NOTHING ELSE ON THE PLANET.

## 🎨 VERDICT SYSTEM

### VERDICT LEVELS
- **✅ PASSED**: SCORE ≥ 90, NO CRITICAL OR HIGH ISSUES
- **⚠️ WARNING**: SCORE ≥ 70, SOME HIGH/MEDIUM ISSUES
- **❌ FAILED**: SCORE < 70 OR ANY CRITICAL ISSUES

### RISK LEVELS
- **🔴 CRITICAL**: IMMEDIATE ACTION REQUIRED
- **🟠 HIGH**: ADDRESS SOON
- **🟡 MEDIUM**: SHOULD BE FIXED
- **🔵 LOW**: NICE TO HAVE
- **⚪ INFO**: INFORMATIONAL ONLY

## 🔧 TROUBLESHOOTING

### COMMON ISSUES

**ISSUE**: JSON DECODE ERROR
- **SOLUTION**: CHECK YOUR JSON FILE FOR SYNTAX ERRORS. USE A JSON VALIDATOR.

**ISSUE**: FILE NOT FOUND
- **SOLUTION**: ENSURE YOUR JSON FILES ARE IN THE `DATASETS/` FOLDER.

**ISSUE**: IMPORT ERRORS
- **SOLUTION**: RUN FROM THE PROJECT ROOT: `PYTHON MAIN.PY` (NOT `PYTHON SRC/MAIN.PY`)

**ISSUE**: PERMISSION DENIED ON LOGS
- **SOLUTION**: CHECK WRITE PERMISSIONS ON THE `LOGS/` DIRECTORY.
